## Introduction

It provides

* **hash**.


* **PRNG** that pass BigCrush and PractRand.


* **APPROXIMATE Gaussian distribution**.


* **Fast range integer RNG on [0,k) credit to Daniel Lemire**.

## Usage

Add this to your `Cargo.toml`:
```toml
[dependencies]
omango-wyhash = "0.1.2"
```
